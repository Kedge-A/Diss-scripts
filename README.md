Opening the folder in MATLAB and running the main.m file will open the GUI.
You can change the quantum well length, x, y, compounds in the heterojunction.
You can save results to an excel file in the folder and pull the last saved results.
You can also save an image of the the graph that is displayed.
There are buttons on the GUI for these functions.
Depending on what the compounds chosen are, sometimes the save results file will fail because of some weird array stuff so just change it by removing or adding {1} to the variable that throws the error.
